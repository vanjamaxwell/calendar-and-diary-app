# 📅 Calendar Diary & Task Manager App

A simple web-based Calendar Diary application built using HTML, CSS, and JavaScript.

It allows users to:
- Select dates from a calendar
- Write and save daily diary notes
- Create and manage to-do tasks per day
- Store all data locally in the browser using localStorage

---

## ✨ Features

### 📅 Calendar
- Monthly calendar view
- Navigate between months
- Highlights today's date
- Click any date to select it

### 📝 Diary System
- Write daily notes per date
- Save notes automatically in browser storage
- Load saved notes when revisiting a date
- Clear notes anytime

### ✅ Task Manager
- Add tasks per selected date
- Delete tasks individually
- Each date has its own task list
- Persistent storage using localStorage

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- localStorage API
